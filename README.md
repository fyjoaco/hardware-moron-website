# Hardware Morón - Sitio Web

## 🇪🇸 Español

Este repositorio contiene la página web oficial de **Hardware Morón**, un servicio técnico especializado en:

* Reparación de celulares
* Reparación de PC
* Reparación de consolas
* Soporte técnico general

## Objetivo del sitio
Brindar información clara sobre los servicios ofrecidos, facilitar el contacto con clientes y mostrar la presencia digital del negocio.

## Tecnologías utilizadas
* HTML
* CSS
* JavaScript

## Funcionalidades
* Página informativa del negocio
* Sección de servicios
* Información de contacto
* Diseño adaptable (responsive)
